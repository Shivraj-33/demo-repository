// how to use git-github step1 :- create an account on github.com then create one repository step2 :- download git for your system then open it and check it is properly install or not with git --version then run 2 commands :- (1) git config --global user.name "your git-hub username" (2) git config --global user.email "your mail that are used for github" step3 :- then open vscode then open terminal on it and then connect your github using git clone your_git-hub_https_link step4 :- then change directory using cd and go to repository main directory then use ls -a cmd to check your .git file is present then step5 :- use git status to check your files are upto-date or not if not then you get 3 errors as follow :- (1) file modified :- present file edited (2) untracted :- means file are new for git-hub (3) unchanged :- all files are upto date step6 :- use add and commit add :- syntax : git add "file_name" / git add . ( used to add all files at one time) commit :- save the recored of changes syntax : git commit -m "any msg related to changes" step7 :- to push all the add and commit data to github repository use :- git push origin main
